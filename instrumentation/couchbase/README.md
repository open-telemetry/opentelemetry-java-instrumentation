# Settings for the Couchbase instrumentation

| System property                                               | Type    | Default | Description                                                             |
|---------------------------------------------------------------|---------|---------|-------------------------------------------------------------------------|
| `otel.instrumentation.couchbase.experimental-span-attributes` | Boolean | `true`  | Enables the capture of experimental span attributes (default is `false` under `otel.instrumentation.common.v3-preview=true`). |
