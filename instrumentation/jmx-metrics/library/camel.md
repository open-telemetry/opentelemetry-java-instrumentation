# Camel Metrics

Here is the list of metrics based on MBeans exposed by Camel.

| Metric Name                                             | Type          | Unit       | Attributes                                                     | Description                                                                                                                                                                                                                                                       |
|---------------------------------------------------------|---------------|------------|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| camel.context.exchange.completed                        | Counter       | {exchange} | camel.context                                                  | Indicates the total number of exchanges processed successfully since context start-up or the last reset operation.                                                                                                                                                |
| camel.context.exchange.count                            | Counter       | {exchange} | camel.context                                                  | Indicates the total number of exchanges, passed or failed, processed since context start-up or the last reset operation.                                                                                                                                          |
| camel.context.exchange.failed.count                     | Counter       | {exchange} | camel.context                                                  | Indicates the total number of exchanges that failed to process since context start-up or the last reset operation.                                                                                                                                                |
| camel.context.exchange.failed.handled                   | Counter       | {exchange} | camel.context                                                  | Indicates the number of exchanges failed and handled by an ExceptionHandler in the context.                                                                                                                                                                       |
| camel.context.exchange.inflight                         | UpDownCounter | {exchange} | camel.context                                                  | Indicates the number of exchanges currently transiting the context.                                                                                                                                                                                               |
| camel.context.exchange.processing.duration.last         | Gauge         | s          | camel.context                                                  | Indicates the time it took to process the last exchange.                                                                                                                                                                                                          |
| camel.context.exchange.processing.duration.last_delta   | Gauge         | s          | camel.context                                                  | Indicates the difference of the Processing Time of the last two exchanges transited the context.                                                                                                                                                                  |
| camel.context.exchange.processing.duration.max          | Gauge         | s          | camel.context                                                  | Indicates the longest time to process an exchange since context start-up or the last reset operation.                                                                                                                                                             |
| camel.context.exchange.processing.duration.mean         | Gauge         | s          | camel.context                                                  | Indicates the mean processing time for all exchanges processed since context start-up or the last reset operation.                                                                                                                                                |
| camel.context.exchange.processing.duration.min          | Gauge         | s          | camel.context                                                  | Indicates the shortest time to process an exchange since context start-up or the last reset operation.                                                                                                                                                            |
| camel.context.exchange.processing.duration.sum          | Counter       | s          | camel.context                                                  | Indicates the total processing time to process all exchanges since context start-up or the last reset operation.                                                                                                                                                  |
| camel.context.exchange.redelivered.count                | Counter       | {exchange} | camel.context                                                  | Number of exchanges redelivered (internal only)  since context start-up or the last reset operation.                                                                                                                                                              |
| camel.context.exchange.redelivered.external             | Counter       | {exchange} | camel.context                                                  | The total number of all external initiated redeliveries (such as from JMS broker) since context start-up or the last reset operation.                                                                                                                             |
| camel.context.route.added                               | Counter       | {route}    | camel.context                                                  | Indicates the total number of routes added successfully since context start-up or the last reset operation.                                                                                                                                                       |
| camel.context.route.started                             | UpDownCounter | {route}    | camel.context                                                  | Indicates the number of routes started successfully since context start-up or the last reset operation.                                                                                                                                                           |
| camel.processor.exchange.completed                      | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Indicates the total number of exchanges the selected processor has processed successfully since processor start-up or the last reset operation.                                                                                                                   |
| camel.processor.exchange.count                          | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Indicates the total number of exchanges, passed or failed, that the selected processor has processed since processor start-up or the last reset operation.                                                                                                        |
| camel.processor.exchange.failed.count                   | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Indicates the total number of exchanges that the selected processor has failed to process since processor start-up or the last reset operation.                                                                                                                   |
| camel.processor.exchange.failed.handled                 | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Indicates the number of exchanges failed and handled by an ExceptionHandler in the context.                                                                                                                                                                       |
| camel.processor.exchange.inflight                       | UpDownCounter | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Indicates the number of exchanges currently transiting the processor.                                                                                                                                                                                             |
| camel.processor.exchange.processing.duration.last       | Gauge         | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the time it took the selected processor to process the last exchange.                                                                                                                                                                                   |
| camel.processor.exchange.processing.duration.last_delta | Gauge         | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the difference of the Processing Time of the last two exchanges transited the selected processor.                                                                                                                                                       |
| camel.processor.exchange.processing.duration.max        | Gauge         | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the longest time to process an exchange since processor start-up or the last reset operation.                                                                                                                                                           |
| camel.processor.exchange.processing.duration.mean       | Gauge         | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the mean processing time for all exchanges processed since processor start-up or the last reset operation.                                                                                                                                              |
| camel.processor.exchange.processing.duration.min        | Gauge         | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the shortest time to process an exchange since processor start-up or the last reset operation.                                                                                                                                                          |
| camel.processor.exchange.processing.duration.sum        | Counter       | s          | camel.context, camel.route, camel.processor, camel.destination | Indicates the total processing time to process all exchanges since start-up or the last reset operation.                                                                                                                                                          |
| camel.processor.exchange.redelivered.count              | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | Number of exchanges redelivered (internal only)  since selected processor start-up or the last reset operation.                                                                                                                                                   |
| camel.processor.exchange.redelivered.external           | Counter       | {exchange} | camel.context, camel.route, camel.processor, camel.destination | The total number of all external initiated redeliveries (such as from JMS broker) since processor start-up or the last reset operation.                                                                                                                           |
| camel.route.exchange.completed                          | Counter       | {exchange} | camel.context, camel.route                                     | Indicates the total number of exchanges the route has processed successfully since route start-up or the last reset operation.                                                                                                                                    |
| camel.route.exchange.count                              | Counter       | {exchange} | camel.context, camel.route                                     | Indicates the total number of exchanges, passed or failed, that the route has processed since route start-up or the last reset operation.                                                                                                                         |
| camel.route.exchange.failed.count                       | Counter       | {exchange} | camel.context, camel.route                                     | Indicates the total number of exchanges that the route has failed to process since route start-up or the last reset operation.                                                                                                                                    |
| camel.route.exchange.failed.handled                     | Counter       | {exchange} | camel.context, camel.route                                     | Indicates the number of exchanges failed and handled by an ExceptionHandler in the route.                                                                                                                                                                         |
| camel.route.exchange.inflight                           | UpDownCounter | {exchange} | camel.context, camel.route                                     | Indicates the number of exchanges currently transiting the route.                                                                                                                                                                                                 |
| camel.route.exchange.processing.duration.last           | Gauge         | s          | camel.context, camel.route                                     | Indicates the time it took the route to process the last exchange.                                                                                                                                                                                                |
| camel.route.exchange.processing.duration.last_delta     | Gauge         | s          | camel.context, camel.route                                     | Indicates the difference of the Processing Time of the last two exchanges transited the route.                                                                                                                                                                    |
| camel.route.exchange.processing.duration.max            | Gauge         | s          | camel.context, camel.route                                     | Indicates the longest time to process an exchange since the route start-up or the last reset operation.                                                                                                                                                           |
| camel.route.exchange.processing.duration.mean           | Gauge         | s          | camel.context, camel.route                                     | Indicates the mean processing time for all exchanges processed since the route start-up or the last reset operation.                                                                                                                                              |
| camel.route.exchange.processing.duration.min            | Gauge         | s          | camel.context, camel.route                                     | Indicates the shortest time to process an exchange since the route start-up or the last reset operation.                                                                                                                                                          |
| camel.route.exchange.processing.duration.sum            | Counter       | s          | camel.context, camel.route                                     | Indicates the total processing time of all exchanges the selected processed since route start-up or the last reset operation.                                                                                                                                     |
| camel.route.exchange.redelivered.count                  | Counter       | {exchange} | camel.context, camel.route                                     | Number of exchanges redelivered (internal only)  since route start-up or the last reset operation.                                                                                                                                                                |
| camel.route.exchange.redelivered.external               | Counter       | {exchange} | camel.context, camel.route                                     | The total number of all external initiated redeliveries (such as from JMS broker) since the route start-up or the last reset operation.                                                                                                                           |
| camel.threadpool.task.active                            | UpDownCounter | {thread}   | camel.context, camel.route, camel.threadpool.name              | The approximate number of threads that are actively executing tasks.                                                                                                                                                                                              |
| camel.threadpool.task.completed                         | Counter       | {task}     | camel.context, camel.route, camel.threadpool.name              | The approximate total number of tasks that have completed execution. Because the states of tasks and threads may change dynamically during computation, the returned value is only an approximation, but one that does not ever decrease across successive calls. |
| camel.threadpool.task.count                             | Counter       | {task}     | camel.context, camel.route, camel.threadpool.name              | The approximate total number of tasks that have ever been scheduled for execution.                                                                                                                                                                                |
| camel.threadpool.task.queue.size                        | UpDownCounter | {task}     | camel.context, camel.route, camel.threadpool.name              | The number of Tasks in the Task Queue.                                                                                                                                                                                                                            |
| camel.threadpool.thread.count                           | UpDownCounter | {thread}   | camel.context, camel.route, camel.threadpool.name              | The current number of threads in the pool.                                                                                                                                                                                                                        |
| camel.threadpool.thread.limit.lower                     | UpDownCounter | {thread}   | camel.context, camel.route, camel.threadpool.name              | The number of threads that are always kept in the pool, even if they are idle.                                                                                                                                                                                    |
| camel.threadpool.thread.limit.upper                     | UpDownCounter | {thread}   | camel.context, camel.route, camel.threadpool.name              | The maximum possible number of threads in the pool.                                                                                                                                                                                                               |
| camel.threadpool.thread.max                             | Gauge         | {thread}   | camel.context, camel.route, camel.threadpool.name              | The largest number of threads that have ever simultaneously been in the pool.                                                                                                                                                                                     |
